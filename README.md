# Euclidean Rhythm Drum Machine

CmajorとReactを使用した、ユークリッド・リズムに基づいたドラムマシンです。

## 技術スタック

- **Frontend**: Vite + React + TypeScript
- **Audio Engine**: [Cmajor](https://cmajor.dev/)
- **Styling**: Vanilla CSS

## ディレクトリ構成

- `src/audio/`: Cmajorパッチ (.cmajorファイル)
- `src/components/`: Reactコンポーネント
- `src/hooks/`: カスタムフック
- `src/utils/`: リズム生成アルゴリズムなどのユーティリティ

## 開発の始め方

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. 開発サーバーの起動:
   ```bash
   npm run dev
   ```
